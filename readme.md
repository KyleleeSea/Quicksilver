Quicksilver
========================
For CMU TartanHacks S26 AppLovin Track 15kb Mobile Game Track

(link demo video here)

## Setup

1. Make sure to use Node 16.x with npm 8.x
   1. `nvm use` to use the current version in `.nvmrc`
2. `npm i`

## Build / Play

1. `npm run build`
2. ./run.sh dist/game.tar.br
